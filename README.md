# LineBOT旅遊小幫手


## 用途
將自助旅行中的住宿、時間行程表、航班資訊等資訊，以LineBOT方式打包，只要在旅行中有網路可用Line時，便可隨時獲取旅行資訊。


## 開發說明
#### 前置設定、綁定帳號
（詳細請參考 https://github.com/yaoandy107/line-bot-tutorial ）

#### 主菜單功能規劃
* 行程規劃
* 飯店資訊
* 最新天氣
* 美食資訊
* 旅程資訊
* 簡易用語（開發中）

#### 最新天氣功能說明
使用 https://openweathermap.org 提供的API，獲取目前當地天氣資訊，以及接下來12小時內的天氣預報
